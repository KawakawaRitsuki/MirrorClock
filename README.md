# Mirror Clock

for Xperia Arc S

## 概要

ちょっとかっこいい時計さん（ニュース付き）

## 注意

このアプリケーションは画面が反転しています。
通常用途では使えません。
また、作者はこのプレートをどうやって手に入れたのかイマイチわかってません（）

おそらくグレースモークのアクリル板だと思いますが。。。多分。。。
かなり透明度の高い黒い板です（）
